# minibook README

minibook 是一个用于管理和查看笔记的 VS Code 扩展。

## Features

- 管理笔记
- 查看笔记
- 搜索笔记

\!\[管理笔记\]\(images/manage-notes.png\)

> Tip: 使用动画展示扩展功能。

## Requirements

无特殊要求。

## Extension Settings

此扩展贡献了以下设置：

* `minibook.enable`: 启用/禁用此扩展。
* `minibook.theme`: 设置笔记的主题。
